# VoX Wallet

This is the cross-platform entry point to the VoX eco system.

Phone verification
Account Creation
Seed words for backup


## Upcoming Features

install npm 6.1.0 or higher
- npm install -g @angular/cli
- npm install -g ionic cordova
- npm i
- ionic build --prod --aot
